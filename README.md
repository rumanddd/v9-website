# V9 Gastrobar

A single-page website design mockup for V9 Gastrobar, an open-flame kitchen and late-night bar.

## Structure

- `index.html` — page markup and content
- `css/style.css` — styling
- `js/main.js` — ember canvas animation and scroll-reveal effects
- `assets/favicon.svg` — site favicon

## Viewing locally

This is a static site with no build step. Open `index.html` directly in a browser, or serve the folder with any static file server, e.g.:

```
npx serve .
```
